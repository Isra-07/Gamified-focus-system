## Project Introduction

Thrive is an Android application that helps users build concentration habits through a **gamified focus timer**. Gamification means the app turns productive behavior into a rewarding experience. Users who complete focus sessions without switching to distracting apps earn points that unlock progression rewards.

The problem addressed by this project is **phone distraction**, particularly among students who struggle to maintain focus despite knowing they need to study. Many users are unaware of how frequently they get distracted or which apps are responsible.

Thrive addresses this by:
- Tracking app usage during focus sessions
- Visualizing distraction patterns over time
- Motivating improvement through a visible progress tracking system
